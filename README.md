# Interactive Choice Website

A playful and interactive website that makes it impossible to say "No"! Built with modern web technologies and smooth animations.

## Features

- Interactive "No" button that evades the cursor
- Smooth GSAP animations
- Beautiful gradient background
- Particle effects
- Confetti celebration on "Yes"
- Responsive design
- Modern glass-morphism UI
- Social links to creator's profiles

## Technologies Used

- HTML5
- CSS3 (with Tailwind-like utilities)
- JavaScript
- GSAP (GreenSock Animation Platform)
- Google Fonts (Poppins)

## Setup

1. Clone the repository:
```bash
git clone https://github.com/AryaDuhan/Question.git
```

2. Open `index.html` in your browser

## How it Works

- The "No" button will playfully move away when you try to click it
- After multiple attempts, the button becomes smaller and eventually disappears
- The "Yes" button triggers a celebration with confetti
- The background gradient responds to mouse movement
- Social links are available in the footer
